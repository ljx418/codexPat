# Orange Tabby Desktop Pet Action Asset Prompt

Create an asset-ready 4x2 sprite sheet for a desktop pet.

Subject:
- Same orange tabby cat in every pose.
- Consistent identity: amber eyes, white chest and chin, pink nose, fluffy orange tabby stripes.

Style:
- Polished semi-realistic cute game asset.
- Clean cutout look.
- Soft shadows.
- Plain near-white background that is easy to remove.
- No text, no labels, no watermark.
- No rounded cards or panel borders.
- Generous spacing between poses.
- Square image.

Eight full-body poses in order left to right, top row then bottom row:
1. idle: sitting calmly, front-facing, tail curled.
2. thinking: sitting, head tilted, curious.
3. running: playful running or leaping forward.
4. success: excited, playing with feather wand toy.
5. warning: alert crouch, ears forward, tail up.
6. error: cute confused expression, gently tangled with a red yarn ball.
7. need_input: sitting with one paw raised toward viewer.
8. sleeping: curled up asleep.

Asset constraints:
- Make silhouettes readable at 64x64 and 128x128 sizes.
- Keep all poses within similar scale.
- Avoid cropped ears, paws, or tail.
- Keep the cat identity visually consistent across all poses.
