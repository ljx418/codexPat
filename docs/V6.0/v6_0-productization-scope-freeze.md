# V6.0 Productization Scope Freeze

status: frozen

date: 2026-05-30

## Frozen Scope

V6 is a productization track. It converts accepted local capabilities into a more understandable, installable, diagnosable, and auditable macOS developer workflow.

Frozen V6 tracks:

1. Release / Distribution Foundation.
2. Codex Work-Cat Product UX.
3. Asset Manager Product UX.
4. Photo-Guided Personalization.
5. Renderer / Visual QA Hardening.
6. Agent / Third-party Developer Productization.
7. QA / Safety / Claim Governance.

## Baseline Rule

V5.12-V5.15 are not reopened as incomplete V6 phases. Their evidence is used as baseline input for V6 product UX hardening.

## Stop Rules

Stop and return to planning if a proposed V6 phase:

- claims production readiness from local smoke.
- treats OS-level discovery as lifecycle monitoring.
- enables provider upload without consent, credential, retention, license, and redaction evidence.
- stores raw photo, EXIF/GPS, full local path, prompt text, or provider raw response.
- expands GLTF prototype evidence to `3D ready`.
