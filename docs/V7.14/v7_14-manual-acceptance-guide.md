# V7.14 Manual Acceptance Guide

status: accepted
date: 2026-05-31

## Minimum Manual Steps

1. Start the desktop app.
2. Create or choose a target PetInstance.
3. Activate the generated 2D action pack.
4. Trigger each core state and confirm the target cat changes visibly.
5. Confirm default and unrelated pets remain unchanged.
6. Activate the imported GLB/GLTF pack.
7. Confirm the 3D cat is visible, nonblank, and in-frame.
8. Trigger action changes or accepted fallbacks.
9. Delete or deactivate each imported pack.
10. Confirm safe fallback or previous active style is restored.

## Required Record

Record pass/fail, screenshots or recording paths, and any observed visual defects. Do not include full local paths, raw provider payloads, prompts, source photos, token, or Authorization values.
