## Conditional Claim Rules

`automatic photo-to-3D ready` is not allowed as a broad claim. A scoped claim may
only be used in this form after V8.2-V8.7 pass:

```text
V8 named-provider photo-to-3D workflow passed for tested <provider>/<model>/<date>
explicit-consent local scenario.
```

The claim must name the provider and must not imply other providers, other
platforms, marketplace readiness, or production release readiness.

## False-green Rules

- MiniMax image generation does not prove 3D generation.
- User-supplied GLB import does not prove photo-to-3D.
- Provider job success does not prove runtime rendering.
- Nonblank screenshot does not prove action-ready 3D.
- Consent boundary does not prove provider integration.
- A single provider smoke does not prove provider integration verified.
- V8.9 local assembly does not prove AI generation.
- V8.10 prompt generation does not prove provider execution.
- V8.11 animated sprite visual QA does not prove 3D, Rive, Live2D, marketplace, or production release readiness.
