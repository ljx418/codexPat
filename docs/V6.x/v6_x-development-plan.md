## Phase Rules

- Every phase requires a development plan, acceptance plan, claim matrix, evidence, final acceptance report, security scan, and claim scan.
- V6 phases may reuse V5 scoped evidence only as baseline; product claims require V6 evidence.
- Provider, production release, 3D readiness, MCP readiness, and cross-platform readiness require independent acceptance.
- QA / Safety / Claim Governance is a cross-cutting requirement for every phase and the V6.9 gate, not a separate readiness shortcut.
