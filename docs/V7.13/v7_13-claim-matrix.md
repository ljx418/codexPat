## Allowed Claim

V7.13 photo-to-asset orchestration passed for tested local 2D generated asset
workflow and external GLB import workflow.

This claim means the guided orchestration layer worked for tested local paths. It
does not mean the app can automatically generate 3D from a cat photo.

## Conditional Claim

V7 automatic photo-to-3D runtime asset workflow passed for tested provider
scenario.

This conditional claim is allowed only if all of the following evidence exists:

- real user-approved photo input or equivalent consented fixture.
- real 3D provider output, not a hand-authored GLB or local fixture.
- explicit provider consent and credential redaction evidence.
- GLTF/GLB deep scan passed.
- import and activation passed for one target PetInstance.
- runtime action mapping passed.
- V7.14 visual QA passed.
