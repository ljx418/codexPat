# V7.11 3D Provider Selection Notes

status: planned
date: 2026-05-31

## Provider Boundary

MiniMax is selected for V7.9 image generation smoke. It is not selected as a GLB/GLTF 3D provider unless later evidence proves supported GLB/GLTF output.

## External 3D Path

V7.11 accepts externally generated GLB/GLTF assets only after local deep scan, license review, and manifest validation.

## Future Provider Candidate Requirements

A future real photo-to-3D provider candidate must provide:

- explicit upload consent flow.
- credential redaction.
- cost, privacy, retention, and license terms.
- GLB/GLTF output or documented export path.
- evidence that raw provider response and user photo are not persisted in project evidence.
