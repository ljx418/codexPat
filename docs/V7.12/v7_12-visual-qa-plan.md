# V7.12 Visual QA Plan

status: planned
date: 2026-05-31

## Visual Metrics

- Nonblank canvas evidence for active 3D renderer.
- Bounding box inside viewport with visible margins.
- Action switch evidence for all core actions or accepted fallback list.
- Scale checks at 1x and 0.75x.
- No transparent-cat state after asset switch, deletion, corrupt file, or restart.

## Evidence Format

Screenshots or recordings must be linked from evidence documents. Evidence must not include full local paths, raw prompts, provider payloads, source photo, token, Authorization, workspace path, or config path.
