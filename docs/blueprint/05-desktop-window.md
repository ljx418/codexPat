# 桌面窗口设计

## 窗口类型

主宠物窗口：

- 透明。
- 无边框。
- 置顶。
- 尺寸随猫咪资产包和缩放配置决定。
- 支持拖拽。
- 位置持久化。

设置窗口：

- 普通窗口。
- 从系统托盘打开。
- 用于配置猫咪资产包、音量、置顶、API、硬件等。

## macOS / Windows 策略

透明无边框：

- Tauri 配置 `transparent: true`。
- Tauri 配置 `decorations: false`。
- 前端根节点保持透明背景。
- MVP 使用猫咪包围盒大小窗口，不使用全屏透明 overlay。

置顶：

- 使用 Tauri always-on-top。
- 设置页允许关闭置顶。
- 不承诺覆盖系统级安全窗口、UAC、部分全屏游戏。

拖拽：

- 按住猫咪区域拖拽窗口。
- 拖拽时暂停自动游走。
- 拖拽结束后保存位置和显示器信息。

点击穿透：

- MVP 不默认支持。
- 后续可作为实验设置：空闲时穿透，用户按快捷键或鼠标进入可交互区后恢复。
- 不在第一版依赖点击穿透实现核心体验。

## 多显示器

规则：

- 启动时读取显示器列表。
- 保存位置如果不在任何显示器内，重置到主显示器右下角。
- 猫咪游走限制在当前显示器 work area 内。
- MVP 不做跨屏自动游走。

边界检测：

- 接近桌面边缘时转向。
- 保留 8 到 24 px margin。
- 窗口缩放或资产包切换后重新 clamp 坐标。

## 系统托盘

菜单：

```text
Agent Desktop Pet
- 显示设置
- 静音 / 取消静音
- 暂停事件 / 恢复事件
- 硬件联动：开 / 关
- 重置位置
- 退出
```

## 设置窗口

MVP 设置项：

- 猫咪资产包选择。
- 音量和静音。
- 置顶开关。
- 事件暂停开关。
- HTTP API 端口。
- API token 查看和重置。
- 硬件联动开关。
- 最近事件日志查看入口。
