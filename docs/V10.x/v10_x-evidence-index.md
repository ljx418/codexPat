## Evidence Rule

V10 evidence must show visible playback behavior. Static screenshots, manifests,
or prompt-only plans cannot replace runtime animation evidence.

V10.4 evidence currently excludes animated GLTF playback claims because no real
animated GLB/GLTF runtime fixture was accepted in this implementation pass.

V10.1-V10.5 passed scoped as the baseline animation pipeline. V10.6 passed
scoped for the local `pet.json` animation format adapter. V10.7 passed scoped
for bundled `work-cat-v1` visual smoke. V10.8 passed scoped for local runtime
micro-interaction controller scenarios. V10.9 passed scoped for Manager
preview/activation polish and does not prove final product-grade acceptance.
V10.10 passed as the scoped product-grade animated 2D work-cat gate. Animated
GLTF runtime playback remains excluded.

V10.11 evidence is screenshot-backed. HTML reports summarize the real desktop
screenshots, but do not replace them as proof.

V10.12-V10.16 are accepted scoped. Their final allowed claim is limited to
tested local macOS visual quality and first-run onboarding scenarios.
