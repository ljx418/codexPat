# V10.12 Open-source Benchmark Spec

status: planned
date: 2026-06-05

## Benchmark Decision

V10.12 compares Agent Desktop Pet against selected public strengths of three
open-source or public desktop-pet references:

- Petdex: public gallery and simple install model.
- OpenPets: ordinary-user assistant setup and verification flow.
- Shijima-Qt: import-oriented desktop-pet library workflow.

The comparison is scoped to two dimensions only:

1. visual quality of local bundled animated pets.
2. first-run onboarding for local work-cat scenarios.

## Surpass Criteria

V10 may say it exceeded the selected benchmark only if:

- the default shipped experience includes at least 6 locally bundled animated
  2D cats.
- every bundled cat has full 8-action coverage and real runtime evidence.
- first-run wizard creates a visible pet in no more than 3 steps.
- first-run wizard creates a Codex work-cat and verifies a reaction in no more
  than 5 steps.
- already-open Codex window automatic monitoring remains clearly unsupported.
- evidence includes screenshots or recordings, not only text.

## Non-goals

- no public remote marketplace.
- no remote pet download.
- no broad ecosystem-size claim.
- no production signed release claim.
- no Windows or cross-platform claim.
- no Petdex parity claim.

## Evidence Requirements

- benchmark comparison table.
- first-run flow screenshots.
- cat-library contact sheets.
- runtime playback captures.
- security and claim scan.
- drawio sync snapshot.
