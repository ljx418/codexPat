## Not Implied

V11 does not imply:

- full Petdex parity.
- ecosystem or marketplace parity.
- broad 3D readiness.
- automatic photo-to-3D readiness.
- provider integration readiness.
- production signed release readiness.
- cross-platform or Windows readiness.
- OS-level Codex binding readiness.
- all Codex workflows verified.

## False-green Rules

- A single moving idle loop does not prove living idle.
- Preview evidence does not prove runtime interaction.
- Click feedback does not prove pointer-aware interaction unless hover,
  double-click, drag, and drop are also tested.
- Demo mode does not prove real Codex state mapping.
- V11 visual action queue must not be claimed as per-instance agent queue.
- Open-source benchmark improvement must be scoped to tested local dimensions,
  not full parity.
